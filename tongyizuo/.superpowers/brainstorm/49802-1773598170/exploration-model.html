How should exploration feel?

Pick the mental model that excites you most. These aren't final — they're directions.

🗺️

A — The Living Map

Start with a small "neighborhood" of ~20 words. As you pan toward the edges, new clusters materialise out of the fog — quietly loading and drifting into place. The whole dictionary is explorable but you only ever see what's nearby.

Feel: Dark Souls bonfire map · Minecraft exploration · No Man's Sky

B — Bloom & Branch

The canvas starts empty except for a single word. Click it → its full cluster blooms outward, showing every synonym and the gloss edges between them. Each outer word has a faint "ring" hinting it has its own cluster. Click any ring → that cluster blooms. Old clusters fade but stay visible.

Feel: The Brain mind-map · Wikipedia rabbit hole · Spore cell stage

🔭

C — Galaxy View + Drill In

Zoomed out: you see tight color-coded constellation blobs — each is a cluster, labeled with its gloss, obviously grouped. Zoom into a cluster → it expands to show individual words, the edges labeled with their connecting gloss. Clusters have gravitational pull on their neighbors, so related clusters orbit each other.

Feel: Google Maps zoom levels · Universe Sandbox · Starfield map
